#### Como usar este GPX da melhor forma (big trail)
**Opção A — Navegação simples**

- Importa o GPX
- Usa os waypoints como “alvos diários”
- Deixa o GPS calcular as estradas entre eles
✅ Ideal se usares Kurviger / Garmin / BMW

**Opção B — Navegação “curvas first”**

- Importa o GPX no Kurviger
- Liga:
	- “Avoid main roads”
	- “Extra curvy”
- Mantém os waypoints fixos
✅ Maximiza prazer de condução

**Opção C — Off‑road separado**

- Usa este GPX como base
- Acrescenta tracks off‑road específicos (se quiseres)
- Mantém as dormidas já marcadas
✅ Muito comum em viagens trail longas

##### ⚠️ Nota importante (transparência)
Este GPX não contém um track contínuo ponto‑a‑ponto, mas sim:

✅ Waypoints corretos e fiáveis
✅ Total compatibilidade com qualquer planeador moderno

👉 Isto é intencional e recomendável para viagens longas:

Evita problemas com estradas fechadas
Permite ajustes diários conforme tempo / cansaço
